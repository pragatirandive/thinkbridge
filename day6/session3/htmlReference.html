CSS Reference

1. Selectors

Element Selector: selects all elements of a type

p { color: blue; }

Class Selector: selects all elements with a class

.highlight { background-color: yellow; }

ID Selector: selects a specific element by ID

#main { font-size: 20px; }

Universal Selector: selects all elements

* { margin: 0; padding: 0; }

2. Colors

JavaScript Reference

1. Variables

JavaScript has three types of variable declarations:

var - Function-scoped variable

let - Block-scoped variable

const - Block-scoped constant

let name = "Pragati";
const age = 22;
var city = "Pune";
2. Data Types

JavaScript has the following data types:

  • String: "Hello"
  • Number: 123
  • Boolean: true/false
  • Undefined: let x;
  • Null: null
  • Object: {name: "Pragati"}
  • Array: [1, 2, 3]
3. Paragraph / Output

Output in JavaScript:

console.log("Hello World");
4. Operators
  • Arithmetic: + - * / %
  • Assignment: =, +=, -=
  • Comparison: ==, ===, !=, !==, >, <
  • Logical: && || !
5. Functions

Function declaration:

function greet(name) {
    return "Hello " + name;
}

Arrow function:

const greet = (name) => "Hello " + name;
6. Loops

For loop:

for(let i=0; i<5; i++) {
    console.log(i);
}

While loop:

let i = 0;
while(i < 5) {
    console.log(i);
    i++;
}
