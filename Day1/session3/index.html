Featured Project

My first featured project is an Online Auction System built using the MERN stack (MongoDB, Express, React, Node.js). This project allows users to register, log in, create auctions, place bids in real time, and view winners when auctions end. It includes features like image uploads with Cloudinary, email notifications, and a cron job that processes auctions automatically.

The goal of this project was to learn full-stack development, improve my understanding of REST APIs, and work with real-time updates. You can explore the source code and documentation on my GitHub once the project is deployed.
